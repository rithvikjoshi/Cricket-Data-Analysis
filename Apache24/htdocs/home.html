Introduction

A Cricket Data Management System is used to find any data about cricket. It can be used by commentators in cricket matches while taking about a stadium or a player. This provides an efficient system for huge data where the time is reduced while getting the data. It provides the tree of data like umpire of the particular match, match referee, player on strike, number of runs scored on that particular ball etc. The user can search with various patterns to get the desired data with high accuracy. The Cricket Data Management System provides a systematic way of searching the data, by which we can assure greater possibility of the required data with analysis of each with every detail required. In order to deal with the future matches the admin can add the required data and set of patterns to get up to date data and several more options to search the data respectively. The user can also suggest various patterns.
